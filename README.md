# Weather App

## 1.Description
A simple weather application that displays the weather forecast for the next 30 days. Users can search for any city to get its weather data.

## 2.Live Demo
[Live demo](https://doniamagdy.github.io/Weather-App/)

## 3.Features

-  Search weather by city name
-  Display 3 days weather forecast
-  Show temperature and weather conditions
-  Responsive design for all devices


## 4.Technologies

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla JS)


## 5.How It Works

1. User enters a city name in the search bar
2. App fetches weather data for that city
3. Displays a 3 days forecast
4. User can search for any other city anytime
